----------小松姫Rainmeterスキンクロック----------

本作品は、「マスコットアプリ文化祭 2015」出展作品です。

小松姫が時間をお知らせしてくれるRainmeterスキンです。
12時版、24時版の2つが同梱されています。

komatsuhime > 12Clock (12時間表示になります。)
komatsuhime > 24Clock (24時間表示になります。)

※ご利用にはRainmeter本体が必要になります。(DL→https://www.rainmeter.net)
